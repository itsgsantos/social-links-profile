## 🚀 Technical Features

* **Mobile-First Workflow:** Optimized for small screens first, ensuring a seamless experience across all devices.
* **Semantic HTML5:** Proper use of structural tags for accessibility and SEO.
* **Interactive Hover States:** Custom CSS transitions for button interactions to enhance user feedback.
* **Clean Typography:** Implementation of professional font-weight and spacing patterns based on the official design guide.
* **Modern CSS Layout:** Utilization of Flexbox for perfect element centering and alignment within the viewport.

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic)
- **Styling:** CSS3 (Flexbox, Custom Properties)
- **Deployment:** Vercel

## 🧠 Key Learnings

Building this profile component allowed me to refine my fundamental front-end skills. 

**Main Takeaways:**
- Implementing **CSS Custom Properties** for efficient color and theme management across the component.
- Mastering **Flexbox centering** techniques for small, high-precision UI modules.
- Organizing a clean folder structure for assets and styles to maintain a professional codebase.
- Perfecting **button styling** and hover effects for a professional, "Apple-style" feel.
